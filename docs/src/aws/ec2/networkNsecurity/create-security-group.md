## 인바운드 규칙

외부에서 EC2로 요청할 때 외부의 요청을 허용할 것인가에 대한 부분 입니다.  
대부분 ssh는 다 열어 놓고 서비스마다 허용하는 내용이 다르니 자신의 서비스를 잘 살펴보시길 바랍니다.

## 아웃바운드 규칙

아웃바운드는 EC2에서 외부로 요청할떄 입니다.  
저는 굳이 막을 필요가 없어서 모두 허용하였습니다.
