## 사용

|        | left | middle | right |
| :----: | :--: | :----: | :---: |
| Header | l/h  |  h/H   |  r/h  |
|        | l/L  |   P    |  r/R  |
| Footer | l/f  |  f/F   |  r/f  |

L, R, H, F를 사용하면 fixed가 됩니다.
